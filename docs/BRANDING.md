# ARC-Emulator Branding

## Primary logo

The primary project logo is stored at:

```text
assets/brand/arc-emulator-logo.svg
```

This SVG is used as the public-facing ARC-Emulator emblem in README files, docs pages, launch posts, and future GitHub Pages surfaces.

## Usage doctrine

- Keep the SVG transparent.
- Do not rasterize it unless a platform requires PNG/JPEG.
- Use the SVG directly for GitHub README and docs where possible.
- Keep generated PNG exports in `assets/brand/generated/` if needed later.
- Do not embed copyrighted game or console branding inside emulator project art.

## Public-facing identity

Recommended brand line:

> ARC-Emulator — Binary-first cryptographic emulator framework for deterministic replay, preservation, and ARC-native game memory.

## Console roadmap lock

```text
N64 → GameCube → PS2 → PSP
```

The logo represents the ARC preservation vessel: a lawful carrier for binary game/session memory, receipts, replay states, and emulator proof lineage.
