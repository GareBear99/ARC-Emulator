# Launch Post Draft

I started ARC-Emulator: a binary-first cryptographic emulator framework for deterministic replay, preservation, and ARC-native game memory. Target order: N64, GameCube, PS2, PSP. The first release focuses on the proof plane: ROM/disc manifests, SHA-256 identity, chunk hashes, Merkle roots, receipts, save-state containers, replay plans, and ARC-Apache integration.
